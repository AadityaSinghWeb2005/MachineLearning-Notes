**how it works :** 
So in this we just create a threshold in the form of percentile and if any value is out of the range then we consider it Outlier 
Eg. Lower limit : 5 percentile and Upper Limit :95 percentile 

After finding all the Outliers There can two methods either we Trim them (remove them ) 
Or we can capping (mean convert them to their Maximum or Minimum Limit )
In this Percentile Method Capping Is called Winsorization
