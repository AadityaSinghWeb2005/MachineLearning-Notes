**Hard Voting VS Soft-Voting**

Hard Voting mein Aap majority vote ka concept use karke final ouput Nikalte hai Ex. Model A -> 1 , Model B -> 1 , Model C -> 0 
Jo Final Ouput Majority ke according 1 Hoga 
Soft Voting Mein Har class ki Probability ki Average nikalte hai Aur jis class ki highest Mean niklta hai Usse Final Ouput Mein dete hai 
Eg. Model 1 -> 1 : 0.40 , 0 : 0.60 
Model 2 -> 1: 0.70 , 0: 0.30 
Model 3 -> 1 : 0.40 , 0: 0.60 
1 Class (probability mean ) : (0.40+ 0.70 + 0.40)/3 = 0.76
0 Class (probability Mean ) : (0.60 + 0.30 + 0.60)/3 = 0.5 

So with soft voting Class 0 will be the final Ouput
