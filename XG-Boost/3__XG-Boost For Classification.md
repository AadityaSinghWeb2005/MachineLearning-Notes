**XG-Boost For Classification**

- **Overview :** Yeh Gradient boosting ke internal Flow ka hi use karta hai jIsme hum base models banate hai Aur har base Model previous base model ki galti ko kam kar rha hota hai. Classification mein first Model mein Log(odds) lenge as a prediction. aur phir Hum second model mein decision tree banaenge with Input columns and Model-1 residuals as Output aur Phir Hum add kardete hai sabhi models ki prediction * Learning rate with first Model Prediction which is just Log(odds). Isme Decision Tree thode alag way mein bante hai 
- **Classification Process Step By Step**

	- **Step-1 :** Hum sabse pehle model-1 banaenge Jiska prediction simple Log(odds) odds ka formula hai : P/1-P Where P is probability of +ve class aur yeh humare Model-1 ki prediction hogi har point ke liye . Ab hum model-1 ke Residuals calculate karenge uske liye sabse pehle hum Model-1 ke prediction ko Probabilities mein change karenge with the hlep of this Formula : $e^log(odds) / (1 + e^log(odds))$ phir bas hum residual calculate karenge by Actual Ouput - Prediciton of Model-1(Probability)
	- **Stage-2 :** Ab stage 2 mein model 2 mein hum ek decision Tree banaenge jisme Input columns aur Model-1 ke residuals as Output jaenge. Ab decision Tree banane ke liye sabse pehle SS(similarity score) Calculate karenge Saare Residuals ke saath isko SS(root) kahenge Classificaiton ke case mein SS calculate karne ka formula hai : $SS = \frac{(\sum \text{Res})^2}{\sum (P \times (1-P)) + \lambda}$ isme Lambda Regularization Term hai P hai Previosu Probability (mtlb uss particular Residual ke liye Model-1 ki Pred(prob)). Ab Hum Data mein main Input column ki help se Splitting Criterias Banaenge Taaki Hum Main root node dhundh sake Aur Goal hoga Similarity score ko maximize karna. maanlo three Potential splitting Criteria Mile : Sp1, sp2 , sp3. Sabse pehle sp1 ke base residuals ko Split karenge Do nodes banenge Left node aur right node Ab Hum left node ks SS nikalenge With same formula and right Node ks SS nikalenge then Sp1 ka Gain nikalenge with thehelp of formula : Left ss + right ss - Root SS aur aise hi hum sp2 , sp3 ka gain nikalenge jisbhi Sp ka gain zada hoga woh Humara root spliting criteria ban jaenga aur Tree banjaega 
	- **Stage-3 :** Ab Hum Tree banke Leaf Nodes ka final Ouput nikalenge with the help of this formula : $Output = \frac{\sum \text{Res}}{\sum (P \times (1-P)) + \lambda}$
	- Ab Model-2 ke prediction nikalne ke liye Hum ISmple Formula ka use karenge which is Model-2 prediction = Model-1 Prediction + Lr* (Model-2 tree Prediction(for particular Input))
